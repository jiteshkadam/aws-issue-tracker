# aws-issue-tracker
Full-stack Issue &amp; Feedback Tracker built with Node.js, React, and AWS (Free Tier)

# Deployment Notes
This project was deployed on AWS EC2 using:
- Amazon Linux 2023
- Nginx reverse proxy
- PM2 for Node.js
- IAM + Session Manager

Infrastructure was terminated after testing to avoid costs.
Can be redeployed anytime from GitHub.
